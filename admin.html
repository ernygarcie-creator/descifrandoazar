Base de captura para actualizar sorteos desde el celular

Este panel no escribe todavía en GitHub por sí solo. Su función actual es capturar, validar y exportar payloads limpios que después correrán contra los scripts de actualización.

Estado de la base
  • Listo: captura de payload, borradores locales, copiar y descargar JSON.
  • Listo: scripts base en scripts/ para validar entradas.
  • Pendiente: backend seguro para escribir al repo y recalcular métricas derivadas.
Formulario
Payload
Cuando conectemos backend, este mismo payload será el que viaje desde admin.html hacia una función segura para actualizar el repo o disparar GitHub Actions.
node scripts/calcular_tris.js --input ruta/al/archivo.json --write
